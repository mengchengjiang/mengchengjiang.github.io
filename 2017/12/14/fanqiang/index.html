那些年翻墙的工具

身在巨大的局域网中，上网总是觉得不自在。翻出去看并不是说可以看到更美好的世界，只是摆脱枷锁，可以自在地在互联网中吸收信息。所以在这里记录一下自己的翻墙经历，也希望这篇之后不用再为翻墙寻找工具。

工具篇

  • VPN
    从一开始就认为免费的vpn是不稳定不靠谱的，所以就直接用了智联科技作为自己首选翻墙工具，也付费了好多年。直到国内不允许公开买卖VPN。VPN也变得各种不稳定，开始了继续寻找爬墙工具的道路。
  • 网页代理
    chome上有很多优秀的翻墙工具，但都前扑后继地慢慢挂掉，一些冷门还没被封的，速度又总跟不上或者干脆不稳定。只能说是鸡肋，作为应急还可以，日常使用还是算了。
  • VPS+shawdowshock
    这是我用过最爽的翻墙工具，可以无压力地在youtube上看1080P视频。因为走的vps流量，所以目前GFW还没做到去封，但据说流量大到被盯上，被封也是有可能的。接下来的内容也主要介绍下怎么在VPS上搭建shawdowShocks。

如何在vps上搭建shawdowshock

首先，你得有个VPS，强烈推荐使用Google目前免费一年的Google Compute Engine。

  • 设置ssh,方便在本地shell连接，Google上的太难用了。
  • 设置外网固定IP。方便配置，不然之后每次重启外网ip变化后，需要对应修改配置中的ip。
  • 设置防火墙。在Google Cloud Platform - 网络 - VPC网络 - 防火墙规则中，设置新建入站，出站规则。
    • IP地址范围 0.0.0.0/0
    • 优先级 1000
    • 协议和端口 all
    • 网络 default

在vps中安装shawdowshock。

  • 使用秋水一键安装版，配置都按照默认，一直按回车就可以。

在Mac安装shawdowshock。

  • 安装
  • chome可以配合SwitchyOmega插件来自动切换，不过SwitchyOmega生效是在PC端的shawdowshock关闭后才生效。

在手机端安装shawdowshock。

  • 没什么好说的，直接安装就可以了。
